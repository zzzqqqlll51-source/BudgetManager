A software try to manage your budget
# 🚀 项目经费管理系统 

这是一个轻量级项目财务管理工具，旨在跟踪多个项目的合同金额、日常支出以及复杂的报销进度。

## 📋 需求分析与功能模块

### 1. 项目档案管理 (Project Profile)
记录项目的身份信息，解决“这笔钱属于哪个项目”的问题。
- **项目全称**：官方正式名称。
- **项目简称**：方便快速输入和搜索的代称。
- **合同总额**：项目最初约定的总金额（预算上限）。
- **项目状态**：进行中、已结项、维护中。

### 2. 日志式支出记录 (Daily Expense Tracking)
随时随地记录每一笔实际发生的消费。
- **基本信息**：日期、金额、经手人。
- **支出类别**：差旅费、招待费（请客）、材料费、劳务费等。
- **关联项目**：这笔钱是从哪个项目里出的。
- **备注**：具体的用途（例如：某某餐厅午餐）。

### 3. 报销进度追踪 (Reimbursement Lifecycle)
这是本项目最核心的功能，追踪钱“回笼”的过程。
- **报销途径**：通过哪个单位或渠道报销。
- **报销状态**：
  - 🔴 **未报销**：钱已花，还没贴票。
  - 🟡 **报销中**：票据已提交，等待到账。
  - 🟢 **已核销**：报销款已全额到账。
- **分批报销管理**：支持一笔大额支出分多次报销，记录每次报销的金额和时间。

### 4. 数据统计与可视化 (Statistics & Reports)
- **项目看板**：实时查看每个项目的：`合同总额` vs `已支出总额` vs `已报销金额`。
- **待处理清单**：一键查询所有“未报销”的明细，防止漏报。

## 🛠 技术路线 (初步设想)
- **开发语言**：Python 3.x
- **交互界面**：Streamlit (Web 界面)
- **数据存储**：SQLite (本地数据库文件) 或 CSV
